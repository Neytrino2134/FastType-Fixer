## üöÄ GETTING STARTED

### Prerequisites
*   **Node.js** (LTS version recommended)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/neon-retro-player.git
    cd neon-retro-player
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # Or simply run 'install.bat' on Windows
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    # Or simply run 'start.bat' on Windows
    ```
